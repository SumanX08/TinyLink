#  TinyLink

A url shortner web application similar to bi.ly 
Built as a take-home assignment using the **MERN stack** (MongoDB, Express, React, Node.js).

Users can shorten long URLs, track click statistics, and manage all their links through a clean and user-friendly interface.

---

##  Features

- Shorten long URLs as tinylink urls
- View detailed click statis 
- Track created date, last visited time & total clicks  
- Single redirect endpoint that also increments click count  

---

##  Tech Stack

### **Frontend**
- React  
- React Router  
- Tailwind CSS  
 

### **Backend**
- Node.js  
- Express.js  
- MongoDB + Mongoose  
- Dotenv  
- CORS
